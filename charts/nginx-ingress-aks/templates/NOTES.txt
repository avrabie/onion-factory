INSTALLATION NOTE:
This chart should be installed using the --create-namespace flag:
  helm install nginx-ingress-aks ./charts/nginx-ingress-aks --namespace {{ .Values.namespace.name }} --create-namespace

To check the status of your ingress controller:
  kubectl get pods -n {{ .Values.namespace.name }}

To get the external IP address of your ingress controller:
  kubectl get service -n {{ .Values.namespace.name }} ingress-nginx-controller -o jsonpath='{.status.loadBalancer.ingress[0].ip}'

IMPORTANT NEXT STEPS:
1. Note the external IP address of your ingress controller from the command above.
2. Configure your DNS to point your domain (e.g., execodex.com) to this IP address.
3. After DNS propagation is complete, install the cert-manager-aks chart to set up TLS:
   helm install cert-manager-aks ./charts/cert-manager-aks --namespace {{ .Values.namespace.name }}

It may take a few minutes for the ingress controller to be ready and for the external IP to be assigned.
